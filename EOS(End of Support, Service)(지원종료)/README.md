# EOS(End of Support / Service)
## End of Support(지원 종료)
- 해당 제품이나 소프트웨어에 대해 더 이상 버그 수정, 보안 패치, 기술 지원 등이 제공되지 않음을 의미한다.
- ex) OS Legacy(Windows 10, Android / iOS 구버전, RHEL(CentOS) 6), Java Legacy(6), Oracle Database 11g, Adobe Flash Player 등 여러 소프트웨어

### EOS의 영향
- 보안 위험 : 지원이 종료된 제품은 보안 취약점이 발견되더라도 패치가 제공되지 않기 때문에 해커나 악성 소프트웨어의 공격에 더 취약해진다.
- 호환성 문제 : 새로운 하드웨어나 소프트웨어와의 호환성 문제가 발생할 수 있다.
- 기술적 지원 부족 : 기술적 문제 발생 시 더 이상 공식적인 지원을 받을 수 없다.
- 비즈니스 리스크 : 특히 기업에서 사용 중인 소프트웨어나 시스템이 EOS 상태가 되면 비즈니스 운영에 문제가 생길 수 있다. 예를 들어, 더 이상 패치가 없으면 시스템 장애가 발생하거나, (보안 취약점 해킹 등을 통해) 데이터 유출이 일어날 수 있다.

## End of Service(서비스 종료)
- 서비스가 더 이상 제공되지 않음을 의미한다. 예를 들어, 클라우드 서비스, 웹 애플리케이션 서비스 등에서 사용되는 용어다. 해당 서비스에 대한 모든 운영, 지원, 기능 개선 등이 종료된다.
- ex) Internet Explorer(iexplore), Kartrider
